# 🎓 Student Info Checker - Python Project

This is a beginner-level Python script that displays student details like name, marks, roll number, and mother's name based on the student's name and roll number input.

## 💡 Features

- Takes input for student name and roll number
- Returns student information (if available)
- Shows an error message if student not found

## 🛠️ Concepts Used

- `input()`
- `if-elif-else` conditions
- Dictionaries
- Basic Python syntax

## 🚀 How to Run

1. Download or clone the repository
2. Make sure Python is installed
3. Open terminal or command prompt
4. Run the program with:

```bash
python student_info.py
